# Contribution au projet PHP Starter Kit

Merci de l'intérêt que vous portez à ce projet ! Les contributions sont les bienvenues.

## Comment contribuer ?

1.  **Signaler un bug** : Ouvrez une "Issue" en décrivant précisément le problème et comment le reproduire.
2.  **Proposer une fonctionnalité** : Ouvrez une "Issue" pour en discuter avant de commencer le développement.
3.  **Soumettre une modification** :
    *   Forkez le dépôt.
    *   Créez une branche descriptive (`feature/ma-fonctionnalite` ou `fix/nom-du-bug`).
    *   Commitez vos changements avec des messages clairs.
    *   Poussez votre branche et ouvrez une Pull Request (PR).

## Normes de code

*   Suivez les standards PSR-12 pour le PHP.
*   Documentez vos nouvelles fonctions/classes.
*   Assurez-vous que votre code ne casse pas les fonctionnalités existantes.

## Processus de Pull Request

*   Une fois la PR ouverte, elle sera revue par un mainteneur.
*   Il est possible que des modifications soient demandées avant la fusion (merge).
*   Une fois validée, elle sera mergée dans la branche principale.

Merci encore pour votre aide !
